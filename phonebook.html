Phone book

Git link

Lien git

Goal

Objectifs

For this project, we had to create a phone book which consists on registering information about companies and collaborators.

Ce projet consiste à créer un annuaire pour entreprises ou l'on enregistre les informations d'une entreprises et de ses collaborateurs

Actions dones

Actions réalisées

I had to create 2 tables, one for the companies and the other for the collaborators.
Then, I had to define roles which allow users to do specifics actions on the website. I had to do a CRUD so people can insert data, show, edit, update and delete.
There is a search button which allows to search datas in function of the name or id.

J'ai du créer 2 tables, une pour les entreprises et l'autre pour les collaborateurs.
Ensuite j'ai du définir des rôles et attribuer des actions pour les utilisateurs. J'ai du créer un CRUD, de cette façon l'admin et les gestionnaires peuvent créer, afficher, mettre à jour voir supprimer les données des tables
Il y'a un bouton de recherche qui permet de rechercher des données en fonction du nom et de l'identifiant.
